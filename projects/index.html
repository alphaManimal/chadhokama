Computational Modeling

Sound Localization, Bilateral Implantation

The focus of my thesis research. A phenomonological model featuring a point process model of the auditory nerve, simplified coincidence detection and interaural time difference tuning to a variety of stimulation regime.

Optimized Defensive Strategies: Algorithms For Defeating the Oregon Ducks

Using linear programming to solve the convex optimization problem, we use a game theory framework to model an efficient decision tree of a defensive end, using the 2011 Oregon Ducks as offense as a training case.
